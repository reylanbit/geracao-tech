## Sobre
- Darlan Saints — desenvolvedor Full-Stack com foco em Java e SQL, autônomo construindo soluções de varejo ponta a ponta.
- Portfólio técnico em SQL, com análises de coortes (LTV), RLS, materialized views e modelagem de dados.
- Projeto público: geracao-tech (este repositório).

## Entregas Técnicas
- Modelagem relacional de app (customers, orders, items, payments, events) com partições e índices.
- Função analítica de LTV por coorte com janelas, LATERAL, recursão e agregações.
- Materialized view indexada com rotina de refresh concorrente.
- RLS simples para eventos por cliente via parâmetro de sessão e role dedicada.
- Portfólio pessoal em schema portfolio (person, skills, projects), dados e consultas de destaque.

## Destaques da Área
- LTV por coortes, retenção e influência via SQL puro.
- RLS para leitura segura por cliente em eventos.
- Materialized views com índices e refresh concorrente.
- Modelagem relacional orientada a varejo e performance.
